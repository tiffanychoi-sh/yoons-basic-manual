📅 매주·매월 정기 체크리스트

매주
  1. 담당 회원 SBT 확인 — 학습시간·녹음·평가 완료 여부 (특히 신규회원)
  2. 가입 3개월 이내 회원 — 주 1회 학습 피드백 문자 발송
  3. 이번 주 시간표 최신본, 상담팀장님과의 카톡 공지사항에 업데이트되어 있는지 확인
  4. 다음 결제일이 임박한 회원 — 교재 재고 부족 여부 확인
  5. 학습계획표 확인 — 수업이 밀린 아이들은 학습계획(달력) 일정을 실제 진도에 맞춰 수정
매월
  1. 3개월 이상 유지 회원 — 월 1회 이상 상담(정착관리) 진행
  2. 전체 담당 회원 시간표 재점검 및 최신화

신규 회원 체크리스트

회원 결제 완료 후, 8단계를 순서대로 진행하세요. 완료 상태는 이 브라우저에 저장됩니다. 하단 [진행상황 복사]로 상담팀장님께 진행 상황을 바로 공유할 수 있어요.

교사 등록 준비 요약

  

    ① 프로필 사진 — 흰 배경·세로·밝은 인상. 스튜디오 또는 집 촬영(골반라인, 밝게 → 본부장님 AI 보정) 선택 가능. 팔짱 포즈 반려!


    ② 자기소개 영상 — 1분 이내, 앞뒤 3초 여유. 연습본 → 상담팀장님 컨펌 → 본촬영.


    ③ 수업 영상 — Greeting → 수업 장면 → 마무리 칭찬. 수업 장면 필수 3요소: 파닉스 게임 2종(smartsam) · 화면공유 자료 2컷(SBT) · 화이트보드(Zoom). 학생 역할 섭외 필요!
